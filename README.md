# 欢迎来到我的主页

我是一个编程领域的初学者，说得好听一点我是一个对整个计算机科学领域中各细化分支都或大或小抱有兴趣的游学者，正蹒跚于这片迷人的山水。我以游玩游戏、开发游戏作为乐趣，我的步伐也不仅限于写代码，有时我会去了解不属于编程开发这块地方的知识。  
严谨而言，我认为自己不恰当冠以"开发者"这一称呼，无论是处于对于我之上的资深大佬们的敬畏也好，还是我希望自己能够综合全能也好，我对自己的定义是一个**创作者**，以制作自己感兴趣的东西为乐趣、动力的**创作者**，无论形式、领域。

# 项目索引

以下陈列了我认为我的那些值得一提的项目，项目主要分为三大类，分别是：非游戏项目、游戏项目，和用于其他游戏的项目(如mod等形式)。

## 非游戏的独立项目

- (停维)[聊天气泡模拟器](https://github.com/bocai-bca/chat_bubble_simulator): 一个Godot4项目，设计于搭配MovieMaker使用。用来模拟聊天软件界面中消息气泡的视觉效果，参考效果是苹果的iMessage。
- [百川HUB](https://github.com/bocai-bca/BaiChuanHUB): 为深海迷航整合包《百川归海》开发的一站式玩家中心，提供整合包安装、卸载与一键启动整合包的功能。
- [简约药物头像横幅](https://github.com/bocai-bca/simple_medicines_avatar_n_backgrounds): 一套简约风格的头像与横幅的Adobe Illustrator工程文件，内容是复刻中国流行的各种药品，主要还原中国常见版本的外观。

- (停维)[终端农场](https://github.com/bocai-bca/TerminalFarm): 一个使用C#编写的控制台程序，是使用命令行交互的种田小游戏，作为C#练手和学习使用Visual Studio IDE而开发的作品。

## 用于其他游戏的项目

### Minecraft

#### 数据包

- [触发器服务系统](https://github.com/bocai-bca/MC_TriggerServiceSystem): 一个框架类数据包，可运行于有"宏"功能之后的MC JE版本，旨在提供自动化侦听和管理触发器记分项，以非常方便地在MC JE原版为其他数据包提供给玩家执行自定义命令的功能。
- [触发器服务系统个人用扩展包](https://github.com/bocai-bca/MC_tss_personal_extension): 我写给自己和朋友一起玩的私人小服使用的TSS的扩展包，里面实现了一些我想要的功能。
- (停维)[矿物蔓延](https://github.com/bocai-bca/MC_MineralGrowing): 一个用于原版生存的数据包，这个包是我为自己和朋友玩而写的，我们不想老是跑图，又担心家附近的矿都被挖完了，于是我写了这个包。本包会在矿物方块上模拟随机刻，使它们会向周围生长。打了本包后你的世界可能会被矿石彻底感染，以至于最后下矿想挖个石头都挖不到。
- (停维)[盾牌攻击守势](https://github.com/bocai-bca/MC_OffensiveGuard): 适用于MC原版生存的玩法改造类数据包。参考了《怪物猎人》系列中的猎人技能"攻击守势"，在MC的盾牌上增加了类似效果。是在和朋友玩原版生存的时候在下界一直被猪灵揍，于是愤怒之下写了这么个数据包，用来战斗爽。
